6.4.3 еӯ—з¬ҰдёІдёҺеӯ—з¬Ұ

еңЁCиҜӯиЁҖдёӯеӯ—з¬ҰдёІжҳҜеӯ—з¬Ұж•°з»„пјҢеңЁLPCиҜӯиЁҖдёӯжІЎжңүеӯ—з¬ҰеһӢеҸҳйҮҸпјҢдҪҶд№ҹеҸҜд»Ҙзұ»дјјеӯ—з¬Ұж•°з»„зҡ„ж–№ејҸеҸ–еӯ—з¬ҰдёІдёӯзҡ„еӯ—з¬ҰпјҢеҰӮпјҡвҖңabcdeвҖқ[0]иҝ”еӣһ97пјҢеҚідёәеӯ—з¬Ұ'a'гҖӮ

CиҜӯиЁҖи§„е®ҡпјҢеңЁжҜҸдёҖдёӘеӯ—з¬ҰдёІеёёйҮҸзҡ„з»“е°ҫпјҢзі»з»ҹйғҪдјҡиҮӘеҠЁеҠ дёҖдёӘеӯ—з¬Ұ'\0’дҪңдёәиҜҘеӯ—з¬ҰдёІзҡ„вҖңз»“жқҹж Үеҝ—з¬ҰвҖқпјҢзі»з»ҹжҚ®жӯӨеҲӨж–ӯеӯ—з¬ҰдёІжҳҜеҗҰз»“жқҹгҖӮеңЁLPCиҜӯиЁҖдёӯзұ»дјјпјҢеӯ—з¬ҰдёІ"china"й•ҝеәҰжҳҜ5пјҢдҪҶе®һйҷ…дёҠеңЁеҶ…еӯҳдёӯеҚ 6дёӘеӯ—иҠӮпјҢ"china"[5]дёҚдјҡжҠҘж•°з»„и¶Ҡз•Ңй”ҷиҜҜпјҢиҖҢжҳҜиҝ”еӣһ0,иҖҢ"china"[6]жүҚдјҡи¶Ҡз•ҢжҠҘй”ҷгҖӮ

иҝҳеҸҜд»ҘдҪҝз”Ё<nд»Һеӯ—з¬Ұз¬Ұз»“жқҹеҸ–еҖјпјҢstr[<x] зӣёеҪ“дәҺ str[strlen(str) - x]пјҢжҜ”еҰӮпјҡ"china"[<1]зӣёеҪ“дәҺ"china[4]"пјҢеӣ дёәеӯ—з¬ҰдёІжҳҜ0еҖјз»“жқҹпјҢ"china"[<0]еҲҷдјҡиҝ”еӣһ0пјҢжүҖд»ҘеҰӮжһңд»Һе°ҫйғЁејҖе§ӢеҸ–з¬¬дёҖдёӘеӯ—з¬ҰжҳҜ<1пјҢеҚі"china[<1]"дёә"a"гҖӮ

6.4.4 еӯҗеӯ—з¬ҰдёІж“ҚдҪң

еңЁLPCиҜӯиЁҖдёӯпјҢеҜ№еӯ—з¬ҰдёІеҸ–еӯҗеӯ—з¬ҰдёІзҡ„ж“ҚдҪңеҪўејҸжҳҜпјҡ

1

str[n1..n2];

е…·дҪ“з”Ёжі•з¬¬дәҢз« е·Із»Ҹд»Ӣз»ҚпјҢзӨәдҫӢд»Јз Ғдёә 2.1.5.cпјҢиҝҷйҮҢеӨҚд№ дёҖдёӢпјҡ

n1е’Ңn2дёәеӯҗеӯ—з¬ҰдёІзҡ„дёӢж ҮиҢғеӣҙпјҢеҜ№й•ҝеәҰдёәnзҡ„еӯ—з¬ҰдёІпјҢn1жңҖе°ҸеҖјдёә0пјҢn2жңҖеӨ§еҖјдёәn-1гҖӮе’ҢеҸ–еҚ•дёӘеӯ—з¬ҰдёҚеҗҢпјҢеӯҗеӯ—з¬ҰдёІдёӢж ҮиҢғеӣҙи¶Ҡз•ҢдёҚдјҡжҠҘй”ҷпјҢиҖҢжҳҜжңүд»ҘдёӢи§„еҲҷпјҡ

  1. еҰӮжһңn1е°ҸдәҺ0пјҢйӮЈд№ҲзӣёеҪ“дәҺ0
  2. еҰӮжһңn2еӨ§дәҺn-1,йӮЈд№ҲзӣёеҪ“дәҺn-1

еҸҰеӨ–иҝҳжңүд»ҘдёӢи§„еҲҷпјҡ

  1. еҰӮжһңn1 > n-1жҲ–n1 > n2пјҢиҝ”еӣһз©әеӯ—з¬ҰдёІ""гҖӮ
  2. е’ҢеңЁеӯ—з¬ҰдёІдёӯеҸ–еӯ—з¬Ұзұ»дјјпјҢеӯҗеӯ—з¬ҰдёІдёӢж ҮеҠ '<‘д»ЈиЎЁд»Һеӯ—з¬ҰдёІе°ҫйғЁејҖе§ӢгҖӮ
  3. еҰӮжһңзӣҙжҺҘеҸ–еҖјеҲ°еӯ—з¬ҰдёІе°ҫпјҢn2еҸҜд»ҘзңҒз•ҘгҖӮеҰӮstr[0..]пјҲжіЁж„Ҹпјҡn1дёҚеҸҜзңҒз•Ҙ)зӣёеҪ“дәҺstr[0..<1]гҖӮ

жҸҗзӨәпјҡиҜ·жіЁж„Ҹstr[n..n]е’Ңstr[n]зҡ„еҢәеҲ«гҖӮ

еӯ—з¬ҰдёІеӨ„зҗҶзӣёе…іеӨ–йғЁеҮҪж•°жҳҜжҜ”иҫғеӨҡзҡ„пјҢеёёз”Ёзҡ„жңүпјҡcapitalizeгҖҒcryptгҖҒexplodeгҖҒimplodeгҖҒlower_caseгҖҒoldcryptгҖҒrepeat_stringгҖҒreplace_stringгҖҒsprintfгҖҒsscanfгҖҒstrcmpгҖҒstringpгҖҒstrlenгҖҒstrsrchгҖҒtrimгҖҒltrimгҖҒrtrim

иҝҷдәӣеҮҪж•°е…·дҪ“з”Ёжі•иҜ·зңӢпјҡhttps://wiki.mud.ren/index.php?title=Lpc:Efun

е…¶дёӯ sscanf() еӨ–йғЁеҮҪж•°еёёз”ЁжқҘеӨ„зҗҶзҺ©е®¶иҫ“е…ҘжҢҮд»ӨпјҢе…·дҪ“иҜӯжі•иҜҙжҳҺеҰӮдёӢпјҡ

еҗҚз§°

1

sscanf() - д»ҺдёҖдёӘеӯ—з¬ҰдёІеҢ№й…Қж јејҸеҢ–иҫ“е…Ҙ

иҜӯжі•

1

int sscanf( string str, string fmt, mixed var1, mixed var2 ... );

жҸҸиҝ°

1

д»Ҙж јејҸ `fmt` еҲҶжһҗеӯ—з¬ҰдёІ `str`пјҢиҝ”еӣһеҖјжҳҜеҢ№й…Қеӯ—з¬ҰдёІзҡ„ж јејҸе®ҡд№үз¬Ұж•°йҮҸгҖӮ`fmt` еҸҜд»ҘеҢ…еҗ«дҪҝз”Ё `%d` жҲ– `%s` еҲҶйҡ”зҡ„еӯ—з¬ҰдёІпјҢжҜҸдёҖдёӘ `%d` е’Ң `%s` жҢүйЎәеәҸеҜ№еә”дёҖдёӘеҸҳйҮҸеҸӮж•° `var1`гҖҒ`var2`...гҖӮ`%d` иҜ»е…Ҙж•ҙж•°еҲ°еҸҳйҮҸпјҢ`%s` иҜ»е…Ҙеӯ—з¬ҰдёІеҲ°еҸҳйҮҸгҖӮ* д№ҹеҸҜд»Ҙз”ЁеңЁж јејҸе®ҡд№үз¬ҰдёӯпјҲеҰӮ %*d е’Ң %*sпјүпјҢиЎЁзӨәеҝҪз•ҘеҜ№еә”зҡ„еҶ…е®№пјҲдёҚеҲҶй…Қз»ҷеҸҳйҮҸпјүгҖӮ

LPCиҜӯиЁҖдёӯзҡ„ sscanf() еҮҪж•°е’ҢCиҜӯиЁҖдёӯзҡ„еҗҢеҗҚеҮҪж•°еҠҹиғҪзұ»дјјпјҢдҪҶеҸҲзЁҚжңүдёҚеҗҢпјҢдёҖж–№йқўдёҚйңҖиҰҒеғҸCиҜӯиЁҖдёҖж ·еҜ№жҷ®йҖҡеҸҳйҮҸдҪҝз”ЁеҸ–ең°еқҖиҝҗз®—з¬Ұ &пјҢеҸҰдёҖж–№йқўеңЁLPCиҜӯиЁҖдёӯ sscanf(str, “%s %s”, str1, str2) дјҡжҠҠз¬¬дёҖдёӘеҚ•иҜҚеҲҶй…Қз»ҷеҸҳйҮҸstr1пјҢеү©дҪҷйғЁеҲҶеҲҶй…Қз»ҷstr2пјҢиҖҢеңЁCиҜӯиЁҖдёӯstr2еҸӘдјҡжҳҜз¬¬дәҢдёӘеҚ•иҜҚгҖӮиҜ·жіЁж„Ҹ %s%s жҳҜдёҚиғҪдҪҝз”Ёзҡ„гҖӮ

еӨ–йғЁеҮҪж•° crypt() з”ЁжқҘеҠ еҜҶеӯ—з¬ҰдёІпјҢеёёз”ЁжқҘеӯҳеӮЁзҺ©е®¶еҜҶз ҒпјҢfluffos е’Ң mudos зҡ„еҠ еҜҶж–№ејҸдёҚеҗҢпјҢеҰӮжһңиҰҒе…је®№пјҢеҸӘиғҪдҪҝз”Ё oldcrypt() еӨ–йғЁеҮҪж•°еҠ еҜҶгҖӮе…·дҪ“иҜӯжі•иҜҙжҳҺеҰӮдёӢпјҡ

еҗҚз§°

1

crypt() - еҠ еҜҶдёҖдёӘеӯ—з¬ҰдёІ

иҜӯжі•

1

string crypt( string str, string seed );

1

еҰӮжһңд»Ҙеӯ—з¬ҰдёІ str еҠ еҜҶеҗҺзҡ„еӯ—з¬ҰдёІеҒҡдёәз§Қеӯҗз»§з»ӯеҜ№еӯ—з¬ҰдёІ str еҠ еҜҶпјҢиҝ”еӣһеӯ—з¬ҰдёІиҝҳжҳҜиҝҷдёӘз§ҚеӯҗгҖӮз”Ё crypt() еҮҪж•°йӘҢиҜҒеҜҶз Ғзҡ„зӨәдҫӢпјҡ
